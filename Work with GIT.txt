1. Создание клона рабочей директории репозитория:
директория-контейнер/git clone <URL нужного репозитория>

2. ДОбавление файла в закладку git:
рабочая директория/git add <имя файла с расширением>

3. Организация закладки:
рабочая директория/git commit -m "<комментарий закладки>"

4. Залитие закладки в удаленный репозиторий:
рабочая директория/git push origin master
